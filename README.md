## 📋 Overview

This project displays a user's test results alongside a summary of category scores. The layout is fully responsive and adapts to desktop and mobile devices.

---

## ✨ Features

- Responsive design
- Modern card layout
- CSS Flexbox
- Linear gradients
- Circular score section
- Hover effect on button
- Summary items with icons
- Mobile-first responsive layout
